CSS Positioning

In CSS, the position property is used to set the positioning of an element on the webpage. There are four values for the position property:

  • Static: The default value. The element is positioned according to the normal flow of the document.
  • Relative: The element is positioned relative to its normal position.
  • Absolute: The element is positioned relative to its nearest positioned ancestor or the initial containing block (the body element), whichever is closer. The closest positioned ancestor refers to the nearest parent element that has a position value other than "static".
  • Fixed: The element is positioned relative to the browser window and does not move when the page is scrolled.

CSS Display Property

The display property specifies the display behavior of an element. There are several values that this property can take:

  • inline
  • block
  • inline-block
  • flex
  • inline-flex
  • grid
  • table
  • table-row
  • table-cell
