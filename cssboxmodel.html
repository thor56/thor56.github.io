CSS Box Model Tutorial

The CSS box model describes how elements on a web page are rendered as a box. This box is made up of four parts: the content, padding, border, and margin.

Content Box

The content box is where the text and images inside an element are displayed. To change the size of the content box, you can use the width and height properties.

This is the content box.

Padding Box

The padding box is the space between the content and the border. To add padding to an element, you can use the padding property.

This is the padding box.

Border Box

The border box is the area around the content and padding. To add a border to an element, you can use the border property.

This is the border box.

Margin Box

The margin box is the space between the border and the next element on the page. To add margin to an element, you can use the margin property.

This is the margin box.
