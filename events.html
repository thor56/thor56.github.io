JavaScript Events Tutorial

This webpage demonstrates how to use JavaScript events to make your webpages interactive.

Example 1: OnClick Event

Click the button to change the text in the Output box.

Code:

<button onclick="changeText()">Click me</button>
<p id="example1">This is some text.</p>
<script>
function changeText() {
  var para = document.getElementById("example1");
  para.innerHTML = "<em>This is italicised text.</em>";
}
</script>

Output:

This is some text.

Explanation:

  1. <button onclick="changeText()">Click me</button>

    This line creates a button element with the text "Click me" and attaches an event listener (onclick) to it. Specifically, it listens for a click event and when the button is clicked, it will call the function changeText().

  2. <p id="example1">This is some text.</p>

    This line creates a paragraph element with the text "This is some text." and an id of "example1". The id attribute is used to uniquely identify this element so it can be manipulated with JavaScript.

  3. function changeText() { }

    This block of JavaScript code defines the function changeText() which is called when the "Click me" button is clicked.

  4. var para = document.getElementById("example1");

    The getElementById() method is a built-in method of the document object in JavaScript. It allows you to select an element in the HTML document based on its unique ID attribute value. In this case, the ID "example1" belongs to the p element that is to be manipulated using JavaScript. The getElementById() method is used to retrieve a reference to the p element, and it is stored in the variable para for further manipulation.

  5. para.innerHTML = "<em>This is italicised text.</em>";

    It is modifying the content of the HTML element with the ID of "example1" that was previously assigned to the variable p. The .innerHTML property of an HTML element allows you to get or set the HTML content inside an element. In this case, the line is setting the innerHTML property of the p element to a new string that contains an HTML em element with the text "This is italicised text." As a result of this line of code, the text "This is some text." that was previously inside the p element will be replaced with the new string containing the em element, which will display the text in italics.

Example 2: OnMouseOver Event

Hover over the text to change the background color:

This is some text.

Code:

<p id="example2" onmouseover="changeColor()">This is some text.</p>
<script>
function changeColor() {
  var para = document.getElementById("example2");
  para.style.backgroundColor = "yellow";
}
</script>

Output:

This is some text.

Explanation:

  1. <p id="example2" onmouseover="changeColor()">This is some text.</p>

    This line creates a p element with the text "This is some text." and attaches an event listener (onmouseover) to it. Specifically, it listens for a click event and when the button is clicked, it will call the function changeText(). The onmouseover event is a mouse event that triggers when the user moves the mouse pointer over the element. This event is set to call the changeColor() function, which can be defined in a separate <script> element or in an external JavaScript file.

  2. para.style.backgroundColor = "yellow";

    para is a reference to p element in the webpage, obtained using the document.getElementById() method. style is a property of the HTML element that provides access to its styling information. It is an object with properties that correspond to the various style properties (e.g. color, font-size, background-color) that can be applied to the element.

    backgroundColor is a property of the style object that sets or returns the background color of the element. Therefore, para.style.backgroundColor = "yellow"; sets the background color of the HTML element referred to by para to yellow.

Example 3: OnLoad Event

The text should change when the page loads:

Code:

<p id="example3"></p>
<script>
document.addEventListener("DOMContentLoaded", function() {
  document.getElementById("example3").innerHTML = "This text was added using JavaScript on page load!";
});
</script>

Example 4: OnSubmit Event

Enter your name and click the "Submit" button:

Code:

<form onsubmit="return greetUser()">
  <div class="form-group">
    <label for="name">Name:</label>
    <input type="text" class="form-control" id="name">
  </div>
  <button type="submit" class="btn btn-primary">Submit</button>
</form>
<p id="example4"></p>
<script>
function greetUser() {
  const name = document.getElementById("name").value;
  document.getElementById("example4").innerHTML = "Hello, " + name + "!";
  return false;
}
</script>
