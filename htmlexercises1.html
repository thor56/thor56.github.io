HTML Tags Examples and Exercises - Part 1

The <h1> to <h6> Tags

The <h1> to <h6> tags are used to define headings:

Heading 1

Heading 2

Heading 3

Exercise 1

Create a heading with the text "My First Heading" using the <h1> tag.

<h1>My First Heading</h1>

The <p> Tag

The <p> tag is used to define paragraphs:

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed euismod nisi auctor, ultrices felis vitae, congue purus. Nulla facilisi. Sed nec dapibus mi. Etiam euismod ullamcorper nunc, id pellentesque mi sagittis non. Aenean volutpat nibh quis enim pulvinar tincidunt. Ut sit amet facilisis dolor.

Exercise 2

Create a paragraph with the text "Lorem ipsum dolor sit amet, consectetur adipiscing elit." using the <p> tag.

<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>

The <a> Tag

The <a> tag is used to define anchors (links):

Visit the W3Schools website.

Exercise 3

Create a link to "https://www.google.com" with the text "Visit Google" using the <a> tag.

<a href="https://www.google.com">Visit Google</a>

The <img> Tag

The <img> tag is used to display images:

Exercise 4

Display the image "https://www.example.com/image.jpg" with the alternate text "Example Image" using the <img> tag.

<img src="https://www.example.com/image.jpg" alt="Example Image">
