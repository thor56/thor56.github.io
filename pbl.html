Project 1

Quotes Machine

Create a webpage that showcases 2 famous quotes. Then gives a choice to the user to add more quotes.

  • Use basic HTML tags like <header>, <nav>, <section>, <ul>, <li>, <blockquote>, <footer>, etc. to create the structure of the webpage.
  • Add some CSS to style the webpage, such as setting a background color, font family, font size, and colors for the different HTML elements. You can also add some hover effects to the links and buttons.
  • Use a bit of JavaScript to add some interactivity to the webpage, such as allowing users to add new quotes or to display a random quote each time the page is loaded.
