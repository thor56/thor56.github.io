HTML Tags and Examples

Simple HTML Tags

Headings

  • <h1> - Defines a top-level heading
  • <h2> - Defines a subheading
  • <h3> - Defines a sub-subheading
  • <h4> - Defines a sub-sub-subheading
  • <h5> - Defines a sub-sub-sub-subheading
  • <h6> - Defines a sub-sub-sub-sub-subheading

Paragraphs

  • <p> - Defines a paragraph

Links

  • <a> - Defines a hyperlink

Images

  • <img> - Defines an image

Lists

  • <ul> - Defines an unordered list
  • <ol> - Defines an ordered list
  • <li> - Defines a list item

Tables

  • <table> - Defines a table
  • <tr> - Defines a table row
  • <td> - Defines a table cell

Line Breaks

  • <br> - a self-closing tag that creates a line break in the text
    • 
                    
    This is some text.
    This is some more text.

Horizontal Rule

  • <hr> - a self-closing tag that creates a horizontal line
    • 
                    
    This is some text.
    This is some more text.

Div & Span

div and span are both container elements that are used to group other elements together. However, they have some key differences in how they are displayed and how they are used.

  • <div> - a block-level element that can be used to contain other elements
  • <span> - an inline element that can be used to contain other elements

HTML 5 Tags

  • <header> - Defines a header for a document or section
  • <nav> - Defines a set of navigation links
  • <section> - Defines a section of a document
  • <article> - Defines an article
  • <aside> - Defines content aside from the content (like a sidebar)
  • <footer> - Defines a footer for a document or section
  • <main> - Defines the main content of a document
  • <figure> - Defines self-contained content, like images with captions
  • <figcaption> - Defines a caption for a <figure> element
  • <video> - Defines a video or movie
  • <audio> - Defines sound or audio content
  • <canvas> - Defines a drawing area
  • <progress> - Defines the progress of a task
  • <meter> - Defines a scalar measurement within a known range
  • <datalist> - Defines a list of pre-defined options for an <input> element
  • <details> - Defines additional details that the user can view or hide
  • <summary> - Defines a heading for the <details> element
  • <time> - Defines a date/time (in machine-readable format)
  • <output> - Defines the result of a calculation or user action
